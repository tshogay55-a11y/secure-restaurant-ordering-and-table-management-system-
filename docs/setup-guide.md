# Setup Guide

## System Requirements
To run the project locally, the following tools are required:

- Node.js
- Git
- Visual Studio Code
- Internet connection

## Installation Steps

1. Clone the repository

git clone https://github.com/tshogay55-a11y/secure-restaurant-ordering-and-table-management-system.git

2. Navigate to the project directory

cd secure-restaurant-ordering-and-table-management-system

3. Install dependencies

npm install

4. Run the application

npm start

5. Open the application in a browser

http://localhost:3000
